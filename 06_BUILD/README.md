# 06_BUILD — build package staging

This folder is for **non-code deliverables** you want versioned next to the app: copy, wedge docs, truth-kernel specs, UI shell references. The app source of truth stays under `src/` and `docs/`.

| Subfolder | Intent |
|-----------|--------|
| `launch_wedge/` | First-ship narrative, positioning, landing copy drafts, checklist exports |
| `truth_kernel/` | Verification rules, receipt/evidence templates, scoring notes tied to `docs/PROJECT-CONTEXT.md` §5 |
| `ui_shell/` | Screenshots, Figma links (as text), component inventory notes for dashboard shells |

Add files as you finalize them; nothing here is required for `npm run build`.
